## 1. このアプリについて

**Bookshelf** は、あなたの本を整理・管理するための個人用ウェブアプリです。

主な特長は次のとおりです。

- **読書ステータスの管理**: 「読みたい」「読んでいる」「読んだ」の3段階で本を整理できます。Booklogのような読書記録機能です。
- **保管場所の記録**: 自宅の書棚、実家、職場など、本がどこにあるかを記録しておけます。複数の場所に本を持っている方に便利です。
- **プロジェクトと参考メモ**: 論文執筆や研究テーマなどのプロジェクトを作成し、本の中から重要な引用やメモをまとめて管理できます。

すべてのデータはあなたのアカウントに紐付けられており、他の人には見えません。

## 2. はじめ方 (アカウント登録・ログイン)

### アカウントを作成する

1. アプリのURL にアクセスします。
2. ログイン画面が表示されたら、「Create account」リンクをクリックします。
3. メールアドレスとパスワード (6文字以上) を入力し、「Sign Up」ボタンを押します。
4. 「Check your email」という画面に切り替わります。登録したメールアドレスに確認メールが届きますので、メール内のリンクをクリックしてアカウントを有効にしてください。
5. アカウントが有効になったら、ログイン画面からログインできます。

### ログインする

1. ログイン画面でメールアドレスとパスワードを入力します。
2. 「Login」ボタンを押すと、本棚のトップページに移動します。

## 3. 本を追加する

### 本の追加を開始する

1. 画面上部のナビゲーションで「Books」をクリックして本棚ページに移動します。
2. 右上の「+ Add Book」ボタンをクリックします。
3. 本の検索画面が開きます。

### Google Books で本を検索する

1. 検索ボックスにタイトル、著者名、またはISBNを入力します。
2. 「Search」ボタンを押すか、Enterキーを押します。
3. 検索結果が表示されます。表紙と書名・著者名を確認して、登録したい本をクリックします。
4. 選択した本の情報 (タイトル・著者・ISBN・表紙画像) が自動的に入力されます。

手動で入力したい場合は、「Cancel」を押して検索をスキップし、タイトルなどを直接入力することもできます。

### 本の詳細を入力する

本を選択すると、詳細入力フォームが表示されます。

| 項目 | 説明 |
|------|------|
| **Title** (タイトル) | 本のタイトルです。必須項目です。 |
| **Author** (著者) | 著者名です。 |
| **Status** (ステータス) | 「Want to read (読みたい)」「Reading (読んでいる)」「Read (読んだ)」から選択します。 |
| **Location** (保管場所) | この本がどこにあるかを選択します。事前に保管場所を登録しておく必要があります (→ [5章参照](#5-保管場所を管理する))。 |
| **Rating** (評価) | 星1〜5で評価できます。星をクリックして評価を付けてください。同じ星を再度クリックすると評価が外れます。 |
| **Review** (レビュー) | 感想や読書メモを自由に入力できます。 |

入力が終わったら「Save」ボタンを押して保存します。「Cancel」を押すとキャンセルされます。

## 4. 本棚を管理する

### 登録した本を見る

「Books」ページを開くと、登録した本がカード形式で一覧表示されます。各カードには表紙・タイトル・著者・ステータス・評価・保管場所が表示されます。

### ステータスで絞り込む

一覧の上部にあるタブボタンで絞り込みができます。

- **All**: すべての本を表示します。
- **Want to read**: 読みたい本だけを表示します。
- **Reading**: 現在読んでいる本だけを表示します。
- **Read**: 読み終わった本だけを表示します。

### 保管場所で絞り込む

タブボタンの右にあるドロップダウンメニュー (「All locations」と書かれたもの) から、特定の保管場所を選ぶと、その場所にある本だけが表示されます。

### テキストで検索する

「Search...」と書かれた入力欄に文字を入力すると、タイトルまたは著者名でリアルタイムに絞り込まれます。

### 本の情報を編集する

1. 編集したい本のカードをクリックします。
2. 編集フォームが開きます。ステータス・評価・レビュー・保管場所などを変更できます。
3. 変更したら「Save」ボタンを押して保存します。

### 本を削除する

1. 削除したい本のカードをクリックして編集フォームを開きます。
2. フォーム左下の「Delete」リンクをクリックします。
3. 確認ダイアログが表示されますので、「OK」を押すと本が削除されます。

削除した本は元に戻せませんのでご注意ください。

## 5. 保管場所を管理する

### 保管場所とは

「保管場所 (Location)」は、本がある物理的な場所を表します。たとえば「自宅の書棚」「実家」「職場のデスク」のように、場所を複数登録しておくと、どの本がどこにあるかを一目で把握できます。

### 保管場所ページへの移動

画面上部ナビゲーションの「Locations」をクリックします。

### 保管場所を追加する

1. 「Add Location」セクションの入力欄に名前を入力します (例: 「東京の自室」「実家の本棚」)。
2. 必要に応じて説明 (Description) も入力できます (任意)。
3. 「Add」ボタンをクリックすると保管場所が登録されます。

### 保管場所を編集する

1. 保管場所の一覧から変更したい場所の「Edit」をクリックします。
2. 名前や説明を編集して「Save」を押します。変更をやめる場合は「Cancel」をクリックします。

### 保管場所を削除する

1. 削除したい保管場所の「Delete」をクリックします。
2. 確認ダイアログが表示されますので、「OK」を押すと削除されます。

保管場所を削除しても、そこに登録されていた本は削除されません。ただし、それらの本の保管場所欄は空欄になります。

### 本に保管場所を割り当てる

本の編集フォームを開き、「Location」ドロップダウンから保管場所を選んで「Save」を押します (→ [3章](#3-本を追加する) および [4章](#4-本棚を管理する) 参照)。

## 6. プロジェクトと参考メモ

### プロジェクトとは

「プロジェクト (Project)」は、論文・研究・勉強テーマなど、目的ごとに参考メモを整理するための入れ物です。たとえば「卒業論文」「機械学習の勉強」「読書感想文」といったプロジェクトを作り、それぞれに関連する本からメモや引用をまとめておくことができます。

### プロジェクトページへの移動

画面上部ナビゲーションの「Projects」をクリックします。

### プロジェクトを作成する

1. 「New Project」セクションにプロジェクト名を入力します。
2. 必要に応じて説明 (Description) も入力できます (任意)。
3. 「Create」ボタンをクリックするとプロジェクトが作成されます。

### プロジェクトを開く

プロジェクト一覧から、開きたいプロジェクトの名前をクリックします。プロジェクトの詳細ページに移動します。

### 参考メモを追加する

プロジェクト詳細ページで操作します。

1. 「Add Reference Note」セクションのドロップダウンから、メモを紐付けたい本を選びます。
2. テキストエリアに、引用文・メモ・気になった内容などを入力します。
3. 「Page ref」欄に参照ページを入力できます (例: 「p.42」「第3章」など)。任意です。
4. 「Add」ボタンを押すとメモが保存されます。

### 参考メモを本ごとに見る

プロジェクト詳細ページでは、追加したメモが本ごとにグループ化されて表示されます。各メモには内容とページ参照が表示されます。

### 参考メモを編集する

1. 編集したいメモの「Edit」をクリックします。
2. フォームにメモの内容とページ参照が読み込まれます。
3. 内容を変更して「Update」ボタンを押します。編集をやめる場合は「Cancel」をクリックします。

### 参考メモを削除する

削除したいメモの「Delete」をクリックします。確認ダイアログで「OK」を押すと削除されます。

### プロジェクトを削除する

プロジェクト一覧ページで、削除したいプロジェクトの「Delete」ボタンをクリックします。確認後、プロジェクトとそれに属するすべての参考メモが削除されます。この操作は元に戻せません。

## 7. ログアウト

画面上部ナビゲーションの右端にある「Logout」ボタンをクリックするとログアウトし、ログイン画面に戻ります。
