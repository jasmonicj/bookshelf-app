## 1. システム概要

Bookshelf は、個人の蔵書を管理するウェブアプリケーションである。
主な機能は以下の通りである。

- **蔵書管理**: 本の登録・編集・削除、ステータス管理（読みたい / 読中 / 読了）
- **場所管理**: 本を保管している場所（自宅・職場など）の管理
- **プロジェクト管理**: 論文・研究などのプロジェクトに紐づけた参照メモの管理
- **書籍検索**: Google Books API を使った書籍情報の自動入力

**技術スタック**

| 項目 | 採用技術 |
|------|----------|
| フロントエンド | Next.js (App Router), TypeScript, Tailwind CSS |
| バックエンド / DB | Supabase (PostgreSQL + Auth) |
| 書籍情報取得 | Google Books API |
| 認証方式 | メール / パスワード認証（Supabase Auth） |

## 2. 共通仕様

### 2.1 認証・認可

- 未認証ユーザーがアプリケーション内ページ（`/`, `/locations`, `/projects`, `/projects/[id]`）にアクセスした場合、ミドルウェアによって `/login` へリダイレクトされる。
- 認証済みユーザーが `/login` または `/signup` にアクセスした場合、`/` へリダイレクトされる。
- セッション管理は Supabase Auth のクッキーベースセッションで行う。

### 2.2 共通ヘッダー（Header コンポーネント）

認証後の全ページに表示される。

```
+------------------------------------------------------------------+
| Bookshelf        [Books]  [Locations]  [Projects]    [Logout]    |
+------------------------------------------------------------------+
```

| 要素 | 内容 |
|------|------|
| ロゴ | 「Bookshelf」テキスト、クリックで `/` へ遷移 |
| Books リンク | `/` へ遷移。現在のパスが `/` の場合はハイライト表示 |
| Locations リンク | `/locations` へ遷移。現在のパスが `/locations` の場合はハイライト表示 |
| Projects リンク | `/projects` へ遷移。現在のパスが `/projects` または `/projects/[id]` の場合はハイライト表示 |
| Logout ボタン | Supabase Auth のセッションを破棄し、`/login` へリダイレクト |

### 2.3 エラー表示

- API エラー・バリデーションエラーは、各フォーム内にインラインで赤文字テキストとして表示する。
- 削除操作の前にはブラウザネイティブの確認ダイアログ（`confirm()`）を表示する。

### 2.4 ローディング状態

- 非同期処理中のボタンは `disabled` 状態になり、ラベルが `...` に変わる。

## 3. データモデル

### 3.1 Book（蔵書）

| フィールド | 型 | 説明 |
|---|---|---|
| id | string (UUID) | 主キー |
| user_id | string | 所有ユーザーID |
| title | string | タイトル（必須） |
| author | string | 著者名 |
| isbn | string \| null | ISBN（Google Books から自動取得） |

| review | string \| null | レビュー・感想テキスト |
| location_id | string \| null | 保管場所 ID（外部キー） |
| created_at | string | 作成日時 |
| updated_at | string | 更新日時 |

### 3.2 Location（場所）

| フィールド | 型 | 説明 |
|---|---|---|
| id | string (UUID) | 主キー |
| user_id | string | 所有ユーザーID |
| name | string | 場所名（必須） |
| description | string \| null | 説明（任意） |
| created_at | string | 作成日時 |

### 3.3 Project（プロジェクト）

| フィールド | 型 | 説明 |
|---|---|---|
| id | string (UUID) | 主キー |
| user_id | string | 所有ユーザーID |
| name | string | プロジェクト名（必須） |
| description | string \| null | 説明（任意） |
| created_at | string | 作成日時 |

### 3.4 ReferenceNote（参照メモ）

| フィールド | 型 | 説明 |
|---|---|---|
| id | string (UUID) | 主キー |
| user_id | string | 所有ユーザーID |
| book_id | string | 紐づく本 ID（外部キー） |
| project_id | string | 紐づくプロジェクト ID（外部キー） |
| content | string | メモ本文（必須） |
| page_ref | string \| null | ページ参照（例: p.42, Ch.3） |
| created_at | string | 作成日時 |
| updated_at | string | 更新日時 |

```
|                                                                  |
|           +--------------------------------------------+        |
|           |                                            |        |

|           |  +--------------------------------------+  |        |
|           |                                            |        |
|           |  Password                                  |        |
|           |  +--------------------------------------+  |        |
|           |  | ••••••••                             |  |        |
|           |  +--------------------------------------+  |        |
|           |                                            |        |
|           |  （エラーメッセージ：赤文字）                 |        |
|           |                                            |        |
|           |  +--------------------------------------+  |        |
|           |  |              Login                   |  |        |
|           |  +--------------------------------------+  |        |
|           |                                            |        |
|           |         Create account                     |        |
|           |                                            |        |
|           +--------------------------------------------+        |
+------------------------------------------------------------------+
```

#### UI コンポーネント

| コンポーネント | 種別 | 説明 |
|---|---|---|
| タイトル | テキスト | 「Bookshelf」 |
| Email フィールド | input[type="email"] | メールアドレス入力。必須。 |
| Password フィールド | input[type="password"] | パスワード入力。必須。 |
| エラーメッセージ | テキスト（赤文字） | ログイン失敗時に Supabase のエラーメッセージを表示 |
| Login ボタン | button[type="submit"] | 処理中は `...` を表示し disabled |

| 機能 | 処理内容 |
|---|---|

#### UI コンポーネント

| コンポーネント | 種別 | 説明 |
|---|---|---|
| ページタイトル | テキスト | 「My Books」 |
| + Add Book ボタン | button | クリックで BookModal（新規登録モード）を表示 |
| ステータスフィルタタブ | button x4 | All / Want to read / Reading / Read。選択中はハイライト |
| 場所フィルタ | select | 「All locations」＋登録済み場所一覧 |
| テキスト検索 | input[type="text"] | タイトル・著者名で部分一致検索 |
| 件数表示 | テキスト | フィルタ後の件数を「N books」と表示 |
| BookCard グリッド | グリッドレイアウト | PC: 3列、タブレット: 2列、スマートフォン: 1列 |
| 空状態テキスト | テキスト | 該当する本がない場合に「No books yet...」を表示 |

#### BookCard コンポーネント

```
+----------------------------+
| [表紙]  タイトル            |
|         著者名              |
|         [ステータスバッジ]   |
|         ★★★☆☆             |
|         @ 場所名            |
+----------------------------+
```

| 要素 | 説明 |
|---|---|
| 表紙画像 | `cover_url` がある場合は画像を表示。ない場合は「No cover」プレースホルダ |
| タイトル | 1行に収まらない場合は末尾省略（truncate） |
| 著者名 | 1行に収まらない場合は末尾省略 |
| ステータスバッジ | want: 黄色、reading: 青色、read: 緑色のバッジ |
| 評価 | ★/☆ で1〜5を表示。rating が null の場合は非表示 |
| 場所名 | `@ 場所名` の形式で表示。場所未設定の場合は非表示 |
| クリック動作 | BookModal（編集モード）を開く |

#### 機能仕様

| 機能 | 処理内容 |
|---|---|
| データ取得 | ページロード時に `books` テーブルと `locations` テーブルを同時取得（更新日時降順） |
| ステータスフィルタ | 選択したステータスと一致する本のみ表示 |
| 場所フィルタ | 選択した場所の `location_id` と一致する本のみ表示 |
| テキスト検索 | タイトルまたは著者名に検索ワードが含まれる本のみ表示（大文字小文字無視） |
| フィルタの組み合わせ | ステータス・場所・テキスト検索は AND 条件で複合フィルタ |
| 保存後の更新 | BookModal の保存・削除完了後、データを再取得して一覧を更新 |

### 5.4 本モーダル（BookModal）オーバーレイ

#### 画面概要

本モーダルは以下の2つの状態を持つ。

| 書籍検索（BookSearch） | Google Books 検索フォームと結果一覧 | 新規登録時の初期状態 |
| 書籍フォーム | タイトル・著者・ステータスなどの入力フォーム | 検索結果から選択後、または既存本の編集時 |

| Delete ボタン | button | 編集時のみ表示。確認ダイアログ後に削除 |
| Cancel ボタン | button | モーダルを閉じる |
| Save ボタン | button | 新規登録または更新を実行。Title が空の場合は disabled |

#### 機能仕様

| 機能 | 処理内容 |
|---|---|
| 書籍検索 | `https://www.googleapis.com/books/v1/volumes?q={query}&maxResults=8` を呼び出す |
| Enter キー検索 | 検索フィールドで Enter キー押下時に検索を実行 |
| 書籍選択 | タイトル・著者・ISBN・表紙 URL をフォームに自動入力し、書籍フォームへ遷移 |
| 新規登録 | `supabase.from("books").insert(data)` を実行 |
| 更新 | `supabase.from("books").update(data).eq("id", book.id)` を実行 |
| 削除 | confirm() 確認後、`supabase.from("books").delete().eq("id", book.id)` を実行 |
| 保存完了 | `onSaved()` コールバックを呼び出し、モーダルを閉じて本棚画面のデータを再取得 |
| ISBN 優先順位 | Google Books の識別子から ISBN_13 を優先して取得。なければ ISBN_10 |
